Work

  • Import Guarantees Management – Desktop App
    A desktop application developed using WPF and C#, designed to manage international import guarantees for the National Bank of Algeria. The application includes several advanced features:
    • User creation and role-based permission transfer, allowing the department head to delegate tasks to other users.
    • Template creation to avoid rewriting standard documents repeatedly.
    • Multilingual support, enabling communication with foreign banks through built-in translation tools.
    • Automatic field filling for recurring guarantee data.
    • Automatic commission calculations based on entered financial data.
    • An integrated chat system allowing employees to communicate without needing to move between offices.
    • A notification system that alerts users when a guarantee is about to expire.
    This project was part of a final-year academic collaboration and aimed at digitalizing and optimizing the workflow of guarantee processing.

  • Ada Lovelace Static Website
    A static website created to highlight Ada Lovelace’s legacy and her pioneering work in computing. The site was developed using only HTML and CSS, with attention to clean layout and visual storytelling.
